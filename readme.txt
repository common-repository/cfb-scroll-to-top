== Description ==

Simply install this plugin and show scroll to top button on your wordpress site....... 

Wanna see how it works? go to screenshot tab on this plugin page on wordpres.
or plugin demo: http://codefairbd.com

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your site and scroll to buttom now you will see a scroll to top button on your theme.

How to Change Scroll To Top Button Icon

Go to `yoursite root dir/wp-content/plugins/our plugin dir/images` now you will see 2 .png images thats called arrow.png and arrowhover.png now replace these image with your selected icons images.(Image size must be '40*40'. Image format must be on png format and must be on same name like(arrow and arrowhover).)
